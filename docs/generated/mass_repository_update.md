## mass repository update

Update an OCI repository's attributes

```
mass repository update <name> [flags]
```

### Options

```
  -a, --attributes stringToString   Replacement custom attributes (e.g. -a owner=data,service=database) (default [])
  -h, --help                        help for update
```
