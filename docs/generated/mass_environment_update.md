## mass environment update

Update an environment's name, description, or attributes

```
mass environment update [environment] [flags]
```

### Options

```
  -a, --attributes stringToString   Replacement custom attributes (e.g. -a environment=staging,region=uswest) (default [])
  -d, --description string          New environment description
  -h, --help                        help for update
  -n, --name string                 New environment name
```
