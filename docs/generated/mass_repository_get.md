## mass repository get

Get an OCI repository by name

```
mass repository get <name> [flags]
```

### Options

```
  -h, --help            help for get
  -o, --output string   Output format (text or json) (default "text")
  -n, --tags int        Number of recent tags to display (newest first) (default 10)
```
