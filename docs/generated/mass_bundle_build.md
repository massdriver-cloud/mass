## mass bundle build

Build schemas and generate IaC files from massdriver.yaml file

```
mass bundle build [flags]
```

### Options

```
  -b, --bundle-directory string   Path to a directory containing a massdriver.yaml file. (default ".")
  -h, --help                      help for build
```
