## mass project update

Update a project's name, description, or attributes

```
mass project update [project] [flags]
```

### Options

```
  -a, --attributes stringToString   Replacement custom attributes (e.g. -a team=ops,system=api) (default [])
  -d, --description string          New project description
  -h, --help                        help for update
  -n, --name string                 New project name
```
